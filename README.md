# SCT_DA_01 - Walmart Sales Dashboard

## Overview
This project focuses on analyzing Walmart sales data using Microsoft Excel. The dataset was explored using Pivot Tables, Pivot Charts, and Dashboard techniques to identify sales patterns, trends, and business insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design

## Dataset
The Walmart Sales Dataset contains weekly sales information from multiple Walmart stores along with factors such as:

- Store Number
- Weekly Sales
- Date
- Holiday Flag
- Temperature
- Fuel Price
- CPI
- Unemployment Rate

## Key Insights

- Non-holiday sales contribute the majority of total revenue.
- Sales peaked during 2011 compared to 2010 and 2012.
- Certain stores consistently generated significantly higher sales than others.
- Holiday periods showed noticeable sales spikes but accounted for a smaller portion of total transactions.
