# SCT_DA_01 - Walmart Sales Analysis Dashboard

## Overview

This project analyzes Walmart sales data using Microsoft Excel. The dashboard was developed using Pivot Tables and Pivot Charts to identify sales trends, compare store performance, and evaluate the impact of holidays on sales.

## Dashboard Preview

![Walmart Sales Dashboard](screenshots/Screenshot%202026-06-12%20095205.png)

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Analysis
- Dashboard Design

## Dataset Information

The dataset contains weekly sales records from 45 Walmart stores along with supporting factors such as:

- Store Number
- Weekly Sales
- Date
- Holiday Flag
- Temperature
- Fuel Price
- CPI
- Unemployment Rate

## Dashboard Components

### 1. Total Weekly Sales by Store

This chart compares the total sales generated by each Walmart store.

![Store Sales](screenshots/Screenshot%202026-06-12%20095310.png)

### 2. Sales Trend by Year

This visualization shows yearly sales performance from 2010 to 2012.

![Yearly Sales Trend](screenshots/Screenshot%202026-06-12%20095219.png)

### 3. Holiday vs Non-Holiday Sales

This chart compares sales generated during holiday weeks and non-holiday weeks.

![Holiday Sales Analysis](screenshots/Screenshot%202026-06-12%20095239.png)

## Key Insights

- Total sales peaked during 2011.
- Non-holiday periods contributed the majority of sales revenue.
- Several stores consistently outperformed others in total sales.
- Holiday weeks showed significant sales activity despite representing a smaller portion of the dataset.
