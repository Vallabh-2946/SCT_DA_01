# SCT_DA_01 - Walmart Sales Analysis Dashboard

## Overview

This project focuses on analyzing Walmart sales data using Microsoft Excel. The objective was to create interactive Pivot Tables and Charts to identify sales patterns, compare holiday and non-holiday sales, and visualize overall business performance through a dashboard.

## Objective

The objective of this task was to:

* Analyze Walmart sales data
* Create Pivot Tables for data summarization
* Generate charts for visualization
* Build an interactive sales dashboard
* Extract meaningful business insights

## Dataset Information

* Dataset Name: Walmart Sales Dataset
* Total Stores: 45
* Features Included:

  * Store
  * Date
  * Weekly_Sales
  * Holiday_Flag
  * Temperature
  * Fuel_Price
  * CPI
  * Unemployment

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Dashboard Design
* Git & GitHub

## Analysis Performed

### 1. Store-wise Sales Analysis

A Pivot Table was created to calculate the total weekly sales generated by each store.

### 2. Holiday vs Non-Holiday Sales Analysis

Sales were compared between holiday and non-holiday periods using a Pie Chart.

### 3. Sales Trend Analysis

A Line Chart was created to observe sales trends over time.

### 4. Dashboard Creation

All visualizations were combined into a single dashboard for better understanding and decision-making.

## Dashboard Insights

* Non-holiday sales contribute the majority of total revenue.
* Sales performance varies significantly across stores.
* Certain stores consistently generate higher sales than others.
* Sales trends provide valuable insights into business performance over time.

Successfully created a Walmart Sales Analysis Dashboard that provides clear insights into store performance, holiday sales impact, and overall sales trends using Microsoft Excel.
