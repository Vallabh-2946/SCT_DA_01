# SCT_DA_01 - Walmart Sales Dashboard

## Overview

This project focuses on analyzing Walmart sales data using Microsoft Excel. The dashboard was created using Pivot Tables, Pivot Charts, and Excel Dashboard techniques to identify sales trends, store performance, and holiday sales patterns.

## Dashboard Preview

![Walmart Sales Dashboard](screenshots/dashboard.png)

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design

## Dataset

The Walmart Sales Dataset contains:

- Store Number
- Weekly Sales
- Date
- Holiday Flag
- Temperature
- Fuel Price
- CPI
- Unemployment Rate

## Dashboard Components

### 1. Total Weekly Sales by Store

Displays total sales generated by each Walmart store.

![Store Sales](screenshots/store_sales.png)

### 2. Sales Trend by Year

Shows yearly sales performance from 2010 to 2012.

![Yearly Trend](screenshots/yearly_sales.png)

### 3. Holiday vs Non-Holiday Sales

Compares holiday and non-holiday sales contribution.

![Holiday Sales](screenshots/holiday_sales.png)

## Key Insights

- Sales peaked during 2011.
- Non-holiday sales contributed the majority of revenue.
- Some stores consistently outperformed others.
- Holiday periods generated significant sales spikes.

## Dashboard Features

- Interactive Pivot Tables
- Dynamic Pivot Charts
- Store-wise Performance Analysis
- Year-wise Sales Trend Analysis
- Holiday Sales Comparison
